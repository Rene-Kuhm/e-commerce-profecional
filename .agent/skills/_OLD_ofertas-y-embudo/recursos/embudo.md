# Embudo de Ventas: DM → WhatsApp

## 1. Atracción (Contenido)
- **Reels/TikTok**: Gancho visual + Propuesta de valor + CTA claro.
- **CTA**: "Comentá [KEYWORD] y te envío info".
- **Objetivo**: Generar el primer contacto (inbound).

## 2. Conexión (DM)
- **Trigger**: Usuario comenta o envía mensaje.
- **Acción**: Respuesta automática o manual rápida.
- **Script**: "¡Hola! Vi que te interesa [KEYWORD]. Para ver si podemos ayudarte, escribime acá: https://tecnodespegue.com/whatsapp"
- **Objetivo**: Sacar al usuario de la red social (donde se distrae) y llevarlo a WhatsApp (donde se cierra).

## 3. Diagnóstico (WhatsApp)
- **Mensaje Inicial**: "Hola, vengo de [IG/TT] por [KEYWORD]."
- **Acción**: Calificación rápida de 5 preguntas (ver `dm-whatsapp-cierre`).
- **Objetivo**: Saber qué necesita, si tiene presupuesto (o idea de inversión) y urgencia.

## 4. Propuesta (WhatsApp)
- **Formato**: PDF de 1 página o Texto estructurado simple.
- **Contenido**: Alcance exacto, precio (rango o fijo si aplica), tiempos.
- **Objetivo**: Que el cliente diga "Sí" o "No" rápido.

## 5. Cierre / Seguimiento
- **Si acepta**: Link de pago (seña 50%) o datos de transferencia.
- **Si duda**: Manejo de objeciones (ver `dm-whatsapp-cierre`).
- **Si desaparece**: Seguimiento 24h / 72h / 7d.

## 6. Entrega y Referidos
- **Cumplir promesa**: Entregar en tiempo y forma.
- **Solicitar testimonio**: Video o captura de pantalla.
